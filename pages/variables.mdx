# Variables

Una variable es un espacio en el cual la computadora almacena un valor. en otras palabras, es una caja en donde guardamos un objeto, para de tal manera siempre conocer el valor, sin la necesidad de referenciarlo constantemente

un ejemplo sencillo es cuando en casa usamos un recipiente para el azúcar y le colocamos una etiqueta con el nombre afuera, cada vez que queramos azúcar buscaremos el recipiente que tenga ese nombre. 

ejemplos:

``` py filename="variables" {0} copy

nombre="Anastasia"

ciudad="traky"

edad= 17

años="17"
 
```

Declarar una variable no es más que el proceso de indicarle a python el valor que guardará en una variable

 *Nota que algunos de estos valores están entre comillas y otros no. Esto se debe a que se usan distintos dipos de datos en programación; en el caso de Python tendremos los siguientes*

## Tipos de datos

* **Str**: representa texto

* **Int**: Números enteros

* **Float**: Números decimales

*Existen otros tipos de datos en Python que estudiaremos más adelante.*

``` py filename="tipos de datos" {0} copy

nombre="Anastasia" ##tipo Str, se escriben entre comillas

nombre2='pueden ser parrafos y oraciones completas' ##pueden ser comillas dobles "" o comillas sencillas''

edad= 17 ##tipo Int, los números se escriben sin necesidad de comillas

precio= 19.23 ## los números(float), decimales, se escriben con .
 
```

### Reglas para nombrar las variables

*1.- no usar palabras reservadas*

*2.- no se puede empezar con números*

*3.- no se pueden usar espacios*
