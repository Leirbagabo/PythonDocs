## Función print 

La función **print()** se utiliza en python para imprimir por consola un mensaje o el resultado de alguna operacion

las funciones tienen un par de parentesis (), dentro de los cuales van los valores con los que va a realizar alguna operacion

``` py filename="print" {1} copy
print("Hola mundo") #va a imprimir el texto Hola mundo por consola
```

``` py filename="print pt2" {3} copy
nombre= "Enzo"

print("Hola", nombre) # Hola Enzo (ya que toma el valor de nombre)
```
*Se pueden enlazar distintos valores dentro de los parentesis separando cada uno con comas "," de esta forma imprimir resultados mas complejos*


``` py filename="print pt3" {5} copy
nombre= "Enzo"

edad= 5

print("Hola mi nombre es", nombre, "y tengo", edad, "años") # Hola mi nombre es Enzo y tengo 5 años
```

*incluso realizar operaciones matemáticas entro del print*

#### Calcular el area de un cuadrado de base 5
